# Survey Module

This folder contains survey data and analysis used to validate hazard predictions.

---

## Contents

- `survey.py` — processing script
- `survey_outputs/` — processed results

---

## survey_outputs/

Includes:
- `cleaned_responses.csv`
- `photo_stats.csv`
- `forecast_class_stats.csv`
- `photo_class_stats.csv`
- `quantitative_stats.csv`
- `text_responses.csv`
- `kincardine_timestamps.csv`

---

## Purpose

- Analyze user perception of water conditions
- Compare predicted hazard levels with observed conditions
- Support calibration of hazard model thresholds

---

## Notes

- Used for validation, not real-time prediction
- Helps ensure model reflects real-world risk perception
